# 🎨 fluxFolio - Frontend Application

This folder contains the frontend workspace for **fluxFolio**, an interactive engineering craft portfolio.

---

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4 (using PostCSS integration)
- **Language**: TypeScript

---

## 🚦 Getting Started

### Development Server
To start the Next.js development server:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the application.

### Build and Production
To build the optimized production package:

```bash
npm run build
npm run start
```

### Linting
To run the ESLint static code analyzer:

```bash
npm run lint
```
