# 🌌 fluxFolio

An interactive engineering craft showcase and portfolio built with next-generation web technologies. **fluxFolio** brings together visual excellence, structural design tokens, and interactive components to present advanced technical work, detailed case studies, and coding challenges.

---

## 🚀 Key Features

- **🎯 Interactive Systems Showcase**: Dive into detailed case studies with interactive syntax highlights, execution metrics, and architecture diagrams.
- **📊 Skill Radar Visualizer**: Interactive vector radar built with custom SVGs showing technical skills and engineering craft.
- **📚 Technical Archives & Blog**: Comprehensive articles on CSS Glassmorphism, Micro-frontends, and React 19 async lifecycles.
- **🎮 Code Games & Quizzes**: Multiple-choice challenges covering React hooks, CSS grid, and advanced TypeScript type theory with explanations.
- **✨ Neo-Futuristic Design**: A cohesive, premium dark-mode theme utilizing glassmorphism, smooth animations, and tailored color gradients.

---

## 🛠️ Technology Stack

| Layer | Technologies Used |
| :--- | :--- |
| **Framework** | [React 19](https://react.dev), [Next.js 16](https://nextjs.org) (App Router) |
| **Styling** | [Tailwind CSS v4](https://tailwindcss.com), Custom CSS variables |
| **State & Logic** | React Context API, TypeScript |
| **Visual Elements** | Canvas API, Interactive SVG animations |

## 🚦 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org) (v18+) and npm installed.

### Setup and Running

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/fluxFolio.git
   cd fluxFolio
   ```

2. **Navigate to the Frontend Directory**
   ```bash
   cd frontend
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Start the Development Server**
   ```bash
   npm run dev
   ```

5. **Open the browser**
   Visit [http://localhost:3000](http://localhost:3000) to view the application in action.

## 🎨 Design System

**fluxFolio** is crafted to deliver a modern, premium aesthetic:
- **HSL Colors**: Cohesive variables using deep slate, indigo, purple, and cyan highlights.
- **Glassmorphism**: Glassmorphic panels implemented with precise `backdrop-filter` rules, subtle boundary borders, and ambient glow shapes.
- **Micro-animations**: Dynamic scale feedback, pulse effects, and smooth layout transitions for active engagement.
